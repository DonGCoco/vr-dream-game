# Anomalist

## Overview

Anomalist is a Unity-based VR/XR interactive experience about layered dreams, fragmented memory, and the unstable boundary between virtual space and physical reality.

This is a 4-person group project uses a surreal apartment-like environment where players explore everyday objects that gradually become abnormal. Through broken logic, hidden spaces, and reality-shifting interactions, Anomalist creates an experience between memory, dream, and waking reality.

## Concept

![Zhuangzi Butterfly Dream](Documentation/zhuangzi-butterfly-dream.jpg)

*Conceptual reference: Zhuangzi’s butterfly dream.*

The game focuses on the relationship between memory, dream, and reality.

Players enter a room that appears familiar and domestic, but the space gradually becomes unstable through interactive objects. Everyday items produce illogical or uncanny feedback, creating the feeling of being inside a fragmented dream.

The project is built around the idea that memory is not a complete recording of the past. Instead, memory often returns as fragments: images, sounds, places, objects, and unclear emotions. These fragments may come from real experience, but over time they can feel as uncertain and distant as dreams.

This idea is reflected in the game’s spatial structure. The player begins in an ordinary room, but each interaction creates a slight disruption in reality. As the experience progresses, the room becomes less stable. Hidden spaces, impossible objects, and surreal events gradually shift the player from observing a domestic environment to questioning whether the space is real, remembered, or dreamed.

The final interaction moves the player toward an XR moment where the virtual layer breaks and the real world begins to appear again. This ending extends the central theme of the project: the boundary between dream and reality is not fixed, but fragile and unstable.

## Key Interactions

The game includes several interactive events:

- A broken cup that does not leak water
- Ducks and wool that create an unsettling gaze-based interaction
- A police car appearing inside the fridge
- A hidden secret room behind a crack
- A butterfly that triggers the ending transition
- Dirty clothes, a board, pizza, a sink, and a mirror as additional interaction points

A visual guide to the object interactions is available here:

[View the Interaction Guide](Documentation/object-interaction-guide.png)

These interactions are not designed to explain one fixed story. Instead, they create a familiar but unstable environment where players can project their own memories, associations, and interpretations.

## Technical Features

This project was developed in Unity with VR/XR interaction support.

Main technical components include:

- Unity scene-based environment construction
- XR interaction setup for controller-based input
- Object-triggered event logic
- Audio feedback and spatial atmosphere
- Animated object responses
- Hidden room and layered environment design
- Passthrough/XR ending concept
- Git LFS support for large Unity assets such as textures, models, audio, and video files
